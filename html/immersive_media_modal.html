    G2GO, Device Care

    Project Background

    Understanding and processing vague and grouped up information/ instructions at once would be an impossible task for most of the students in Grace Orchard School. In addition, some students are unable to read and hence require text-to-speech functions. Sadly, most games do not come with broken down instructions or text-to-speech.

    G2GO, Device Care is an application made with the intention of supporting and assisting teachers, as well as helping students retain and practice device care practices taught to them through the gamification and reinforcement of concepts taught by their teachers. Hence, this application comes with a text-to-speech function, as well as a code teachers can enter to give the student an achievement badge if the student has really understood the device care practice.

    Key Benefits

    Assist - Teachers can use the application to verify if students have understood the device care practice taught

    Motivate - Students will be motivated to do better to get the more attractive achievements out of the 3 possible achievements available

    Key Technologies Used

  • Inkarnate.com – Map Creation
  • Maya – Game Models
  • Photoshop – Sprites/ Images/ Textures
  • PISKELApp.com – Sprites/ Images
  • Unity – Game Engine

    • Potential Enhancements

  • Implement Text-To-Speech for Game Levels as well

Because each department is preoccupied with its own tasks, they do not have time to learn about the work of the other departments. Conducting an online quiz would be straightforward and dull, so the staff would be even less interested as a result.

    REBEE-RRT

    Project Background

    Rebee-RRT is a project that is intended to be sold to the army for to conduct RRT testing remotely. The special functionality of the project is to be able to detect when the person is cheating, while using the sensor to do the exercise. There are multiple warning systems in place to detect when the user is cheating, such as when the user does the exercise too fast, or when the user is in a wrong position when doing the exercise. All these warnings make the results more reliable, which makes the entire system more feasible to the army who will get accurate results from the trainee’s exercises, allowing them to conduct remote testing, preventing the spread of COVID.

    Key Benefits

    Reliability - Results are more accurate and reliable for conducting remote testing.

    Remote - Testing can be conducted remotely, preventing spread of COVID amongst trainees/testers.

    Convenience - Trainees no longer have to come down to the testing area in order to be tested.

    Key Technologies Used

  • Unity Game Engine
  • IOT Gyroscope Sensors

    • Potential Opportunities

    This project can be enhanced with the following:

  • Storing warning data online: Allows for the testers to check what warnings the trainee has triggered to check whether the trainees are actually cheating, and see what they are doing wrong.

    Project Hygge

    Project Background

    Recently, companies are struggling to get their teams to bond, as the COVID-19 pandemic limits in-person social interactions and meet-ups. With the limited amount of team bonding activities that are available, companies are in need of an option that caters to their needs and values.

It features immersive engaging puzzles that are tailored around a customizable set of words, allowing companies to modify it based on their own needs and requirements. Thus, it offers a modular solution that provides a virtual team bonding experience, through a game that companies could use to teach their core values, history, and even more.

    DokiDare - Scene System

    Project Background

    Streaming as a Virtual Tuber has been on the rise and some published application offers such platform for these users to stream their contents using virtual avatars. However, setting up their scenes virtually can prove to be very troublesome as it takes considerable amount of time for them to beautify their scenes, especially when launching the application again.

    DokiDare offers a Scene System feature that helps users to save the data of all the items that exists in the current scene, right down to where the items are placed and able to load them exactly as how it was set up. The loading scenes are also interchangeable which makes it more convenient to load scenes even while streaming.

    With the Scene System implemented, it will eliminate the need to manually set up scenes from scratch whenever users re-launch the application.

    Key Benefits

    Adjective - Eliminate the need to manually set up scenes after every start of application.

    Convenient - Easy to use with unlimited number of scenes to save. Interchangeable between loaded scenes and account-based usage where saved scenes can only be loaded by the user’s account it was saved in.

    Key Technologies Used

  • Unity Game Engine
  • Unity’s JSON Utility

    • Potential Enhancements

    This project can be enhanced with the following:

  • Create an overwrite button to quickly overwrite current loaded scene.
  • Create a clear button that clears all existing items in the scene to restart scene set up.

    AR Balloon

    Project Background

    From birth, children learn to identify, express, and manage their emotions. They learn this ability from significant adults in their lives, such as parents and grandparents. For example, a child's behavior will reflect their perception of anger and associated aggression, such as shouting. Consider being unable to identify emotions at a young age and then being unable to demonstrate or express one's own emotions, as we were taught emotions are for the weak. That describes the majority of us, including you. Therefore, why not assist in educating children about emotions? 

    Emotion Learning Detector is an emotion detection software trained using deep learning technologies (AI and ML).  It has multiple features such as a gamified part of the app and facial emotion detection, where children must scan their surroundings to find and get NPCS to spawn and learn about emotions through real life people or themselves, respectively. This allows them to decide on the choices to make where they can learn how to deal with their own and as well as other's emotions. Thus, this can allow them to learning and better deal with emotions at a young age.

    Key Benefits

    Interesting - Children get excited easily, so by detecting their emotions can interest them

    Convenient - Since it is an app, it can be downloaded in any devices such as Apple and Android

    Educational - Teaching children about the different emotions and how to handle them

    Key Technologies Used

  • Modelling: Maya, Adobe Photoshop, Adobe Illustrator, Unity Asset Store
  • Animation: Animation – Maya
  • Design: Adobe Photoshop, Adobe Illustrator, PowerPoint, Adobe XD, Unity Asset Store
  • Programming: Visual Studios, Unity Asset Store
  • Base: Unity (Version 2021.1.22f1)

    • Potential Opportunities

    This project can be enhanced with the following:

  • Adding more effects on the balloon when it pop
  • Add a better background for the main menu to fit with the UI better

    Personalised Feedback App

    Project Background

    Our Project is to improve the effectiveness of feedback given to FILA (Facts, Ideation, Learning Issue, Action Plan) and SDL (Self-directed learning) by allowing students to view feedback and scores of their FILA and SDL side by side in the same app.

    Viewing the FILA and SDL with the feedback and scores given lessens the effectiveness of the feedback as students will have to take time to compare between their phone and computer. By allowing the feedback and scores to be compared directly side by side the student will be able to know where is the problem faster.

    Key features for the Personalised background app include image links to link websites directly for the teacher to view, image upload sections if the student wishes to upload an image using his camera/phone and a section where the student can view written feedback by the teacher.

    Key Benefits

    Helpful - Students can learn from their mistakes and understand the feedback easily for each of their work.

    Convenient - The app allows students to key in their work and view the scores and feedback on the same page.

    Key Technologies Used

  • Glide App
  • Google Sheets

    • Potential Enhancements

    This project can be enhanced with the following:

  • An upgraded version of Glide which allows storing real email address.
  • A better app development software with lesser restrictions.
