Data Scraping with RPA

Project Background

Currently, information from invoices of the description of construction materials and their prices are manually scraped and recorded into an Excel database. For future projects, the database would allow for the retrieval and comparison of prices of similar materials. However, this manual scraping process is both time-consuming and tedious.

With the implementation of Robotic Process Automation (RPA) using UiPath, the entire process can be automated by mimicking the user's sequence of steps taken to complete the task. The Optical Character Recognition (OCR) technology instantaneously extracts texts from the PDF invoice documents and transfers the information over to the Microsoft SQL database. The data is stored in Microsoft SQL database as it facilitates greater ease of data retrieval using SQL queries, as compared to using Excel.

Key Benefits

  • Efficient – Manual and labor-intensive processes such as data entry are automated at a click of a button.
  • Cost Saving – Usage of Open Source UiPath software allows for cost-savings for the company.
  • Reliable – Allows for data from the invoices to be accurately captured with minimal errors.

Key Technologies Used

  • UiPath
  • Microsoft SQL
  • Microsoft Excel

Potential Enhancements

This application can be enhanced with the following:

  • Using a more complex OCR model that includes handwriting recognition ability.
  • Training the OCR model to recognise different invoice layout types to ensure that information is captured more accurately.

FoodBuy – An E2E Business Solution

Project Background

The COVID-19 pandemic has brought about a massive acceleration of e-commerce. Micro-merchants with minimal or no experience with e-commerce platforms faced an immense struggle trying to make a digital transition that was sustainable. This provided the inspiration for FoodBuy.

FoodBuy is a full end-to-end (E2E) business solution which comprises mobile applications and a website which collectively aids micro-merchants to digitalise their business. It creates a sustainable ecosystem, offering the micro-merchants opportunities to generate more revenue. This E2E business solution operates on a Group-Buy model aimed to solve the pain points faced by the micro-merchants. It serves to drive customer traffic flow for the merchants while remaining sustainable for all users, including the customer and the delivery personnel.

This project received the 1st Runner-up Award for the Sustainable Finance Category Finals of the PolyFinTech100 API Hackathon 2021.

Key Benefits

  • Increased Revenue - Increase in bulk sales of products.
  • Cost-saving - Reduce overall costs incurred when digitalising a business.
  • Seamless digital integration- Launches the end-to-end solution with the click of a button.

Key Technologies Used

  • Ionic 5 Framework (Angular)
  • Firebase Console (Database)
  • Adobe XD (Designing Tool)
  • Visual Studio Code (Source-code Editor)

Potential Enhancements

This mobile application can be enhanced with the following:

  • Additional payment application programming interface (API) to increase choice flexibility for users.
  • Social media plugin to allow users to conveniently share the listed products with others.
  • Additional tips feature for users wanting to give extra support to micro-merchants.

GreenTrek Mobile Application

Project Background

The Singapore Green Plan 2030, or the Green Plan, is a whole-of-nation movement to advance the national agenda on sustainable development, and position Singapore to achieve its long term net zero emissions aspiration as soon as viable.

GreenTrek is a mobile application that helps raise awareness about the Green Plan, educates people on its initiatives and how they can get involved to become informed advocates for nature’s conservation and green spaces.​ Its user-friendly interface provides quick access to expertly-curated information on all the nature parks and reserves. Through this free to download application, people are able to learn about the historical and scientific details of Singapore’s various parks while getting help exploring the many hiking trails, providing them with an engaging new perspective on nature in the “Garden City”.

Interactive Chatbot for AMS

Project Background

Examinations and assignments are a significant part of the education experience in Institutes of Higher Learning (IHLs) like Temasek Polytechnic. As formal tests and project submission deadlines draw near at the end of each semester, frantic students often request for final consultations with their lecturers to seek concept clarification or feedback on completed work. However, fixing appointments to meet with equally busy lecturers continues to prove challenging when many students need help at the same time. The following project provides a solution to this regular semester predicament facing students.

The Interactive Chatbot for Aviation Management System (AMS) is a computer programme that simulates human conversation using text commands and utilises an artificial intelligence (AI) feature that can be embedded on a freeware instant messaging app like Telegram. Students simply post their enquiries using the messaging app and wait for a response from the chatbot. The chatbot ultimately aims to address frequently asked questions (FAQs) in an immediate manner by retrieving answers from a database. Developed using deep learning technologies that aid natural language understanding, the chatbot detects and comprehends the user's intent through the typed text, providing the correct answer in response. For this prototype software, the chatbot is an augmented service only for the Diploma of Aviation Management since AMS students are the ones who provided all the FAQs.

Key Benefits

  • Offers model answers - From lecture notes and past students
  • Fuzzy matching - e.g. Search for “product dev” matches “product development”
  • 24/7 operation - Once turned on, there is no need for maintenance, but may be updated
  • No need for an Internet connection- It can be used offline

Key Technologies Used

  • Dialogflow (natural language understanding [NLP] platform)
  • Firebase (database provided by Google to develop mobile and web applications)
  • JavaScript (backend Integration / helps connect the chatbot to Firebase)
  • Machine Learning (provides training phrases to Dialogflow to learn)

Potential Enhancements

This mobile application can be enhanced with the following:

  • Easier navigation to allow users to navigate back to the previous query
  • Better accuracy in responses
  • Neater database that is well-arranged and categorised
  • More platforms can be deployed to fit different needs

SGenius Student Learning Portal

Remember the time when we had to lug and comb through many files and workbooks that made studying tiring and tedious? How can we make studying fun and more customised? SGenius is a personalised online learning platform that allows students to create customised Mathematics worksheets based on their needs. Students will also be given statistical feedback that enables them to improve in the subject.

SGenius contains hundreds of questions in its database that matches the students’ needs to their ability. It allows students to extract questions from the database based on their individual needs and preference. This customisation combined with a simple yet engaging interface with statistical features allow students to revise in a fun and interactive manner.     TPsecureKeys Key Management System

    Project Background

    Temasek Polytechnic has a sprawling campus of numerous buildings and facilities spread over 300,000 square metres of land. Currently, staff and third-party visitors who reserve facilities do so through a manual system of loaning and returning keys that provide access to the facilities. This borrowing process requires administrative staff to send loan and return forms via email. After which, a manually updated computer spreadsheet tracks the loan status of keys and facility users.

    TPsecureKeys is a key management system application designed with PowerApps and SharePoint. It automates the key loaning and returning process using form and database integration. A submitted Loan and Return form in a PowerApps application linked to SharePoint will automatically create a new entry with the borrower’s details sent to the SharePoint database. Upon return of a key, a submitted key return form will update its availability status in the same database. This application replaces the manual work of issuing hundreds of keys that may exchange hands daily, offering an efficient digitised approach to managing access to Temasek Polytechnic's facilities.

    Key Benefits

    • Cost-efficient - Savings on purchasing database services and ensures keys are always accounted for
    • Convenient - Streamlines the process of loaning and returning keys
    • Automated logging - Transaction log enables better tracking of users issued keys

    Key Technologies Used

    • SharePoint (to create websites to securely store, organise, share, and access information from any device)
    • PowerApps (for building and using custom apps that connect to data and work across the web, and mobile)

    Potential Enhancements

    This application can be enhanced with the following:

    • Implementation of a notification system for updated inventory
    • A more robust search and filter function for tracking accuracy

    Deployment of TEMI Robot for Student Care Centre

    Project Background

    Imagine having only a few teachers to handle a class of over 50 students with different learning abilities, ranging from Primary 1 to Primary 6. Life Student Care aims to solve this problem by integrating a TEMI robot into the student care centre that serves as a virtual teacher.

    TEMI’s in-built Android OS allows for custom and Google Play Store applications to be installed. This creative application allows students to input their learning and mood for the day as well as allows teachers to view and resolve any issue that might surface with the students. The robot also uses TEMI’s in-built function to move around the classroom, bringing it to life. Robots such as TEMI has not been implemented in any school or student care centre, making Life Student Care the pioneer with this innovation, with no other competitors.

    Key Benefits

    • Resourceful - Teachers can complete other tasks while the robot attend to the students
    • Practical - Reduces the impact of manpower shortages as TEMI can help out with some tasks usually done by teachers
    • Convenient - Teachers can have visibility on what their students write through the app, instead of books

    Key Technologies Used

    • Temi Lidar Sensor
    • Android Java
    • Google Cloud Firestore

    Potential Enhancements

    This project can be enhanced with the following:

    • Creating a separate app or website for teachers as administrator instead of implementing it in the same app
    • Incorporating the use of TEMI’s SDK to make the app more interactive

    ASEAN-IFCE Resource Network Portal

    Project Background

    The ASEAN-IFCE Resource Network Portal was created to enable jurisdictions across the ASEAN region to assess and improve the quality of justice and court administration by using the IFCE* and IFJSE* framework implemented by the ICCE*. Its members are individuals who are leaders and high-ranking officials in their own legal jurisdictions. This project aims to create a new website, by including functionalities such as member-exclusive forum and programs page.

    Members will be able to sign-in to the website, gaining access to a forum that allows them to communicate, mingle and network with one another. Additionally, members will gain access to documents before and after each meetup. As this website will be mainly used by jurisdictions in the ASEAN region, a translation service has been included to facilitate members from the ASEAN countries to view the website in their own native language.

    ICCE* – International Consortium for Court Excellence

    IFCE* – International Framework for Court Excellence 

    IFJSE* – International Framework for Judicial Support Excellence 

    Key Benefits

    • Simple and seamless - Easy to manage website using Web GUI
    • Convenient - Connect ICCE members through forums and in-built translation for non-English users

    Key Technologies Used

    • Angular
    • MySQL
    • NodeJS and ExpressJS

    Potential Enhancements

    This project can be enhanced with the following:

    • Implement search engine optimisation (SEO) and send email response utilising APIs such Twilio
    • Translate both dynamic and static content
    • Format and embed information through text box for users to attach information
